Education Support

  • Tech Stack: Android Kotlin, Realtime Firebase, Firebase Authentication
  • Github URL: Project Link

The Android application “Education Support” is developed using Kotlin. The application covers all the features which an educator and learner will require to learn more about courses and take quizzes.

Both educators and learners can register themselves and login securely into the app. Educators can add new courses and new quizzes for those courses. The quizzes could have single or multiple correct answers and support audio questions.Educators could also add learners and, they can send an email via the platform’s native email-sending application. Educators can also see graphical statistics of the results for each quiz.

Learners can search existing courses and can enroll themselves in any course. They can get take a quiz and, on each answer, we display the correct answer. Once completed, they can view the results of the quiz.
