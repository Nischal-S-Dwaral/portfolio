Java Spring Boot and React JS based platform for continuously inspecting documentation comments quality in a Java codebase.

The ‘Find a Service’ web application addresses customers' need to easily locate reliable service providers in their area. This web application serves as a centralized platform for connecting users with trustworthy service providers by offering a diverse range of services including cleaning, babysitting, pest control, plumbing, electrical repairs, beauty etc.

The application emphasizes that all service providers undergo a rigorous screening process by the admin to guarantee customer satisfaction. By streamlining the search process and providing a convenient platform, ‘Find a Service’ provides a valuable solution for users asking for reliable service providers across various industries.
