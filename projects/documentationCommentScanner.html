Documentation Comment Scanner

  • Tech Stack: Java Spring Boot, React JS, Firebase Authentication, Firestore, Firebase Storage
  • Backend GitHub URL: Backend Code
  • Frontend Github URL: Frontend Code

Java Spring Boot and React JS based platform for automated Javadoc scanning and analysis, which quantitatively evaluates documentation comments within Java codebases.

  • User Authentication and Authorisation: Secure user authentication with email and password to prevent unauthorised access using Firebase.
  • Initiating Analysis: Create projects with GitHub configurations and initiate documentation comments analysis for selected projects.
  • Viewing Analysis Results: Display analysis results, including summary metrics and code views for Java files with necessary review comments.
  • Quality Gate: Enforce specific project coverage percentages and update quality gates for projects.
  • Access Control: Ensure all use cases require user authentication.
